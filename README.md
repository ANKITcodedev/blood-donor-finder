# Blood Donor Finder API

A life saving app that connects blood donors
with patients in need across India.

## Live API
**Swagger UI:** https://blood-donor-finder-production.up.railway.app/swagger-ui/index.html

**GitHub:** https://github.com/ANKITcodedev/blood-donor-finder

## Features
- JWT Authentication
- Donor Registration & Management
- Emergency Blood Request System
- Location-based Donor Search (Haversine Formula)
- Email Notifications
- Reward Points & Badge System
- Auto Scheduler
- Swagger API Documentation

## Tech Stack
- Java 17, Spring Boot
- MySQL, Redis
- JWT, Spring Security
- Railway (Deployment)

## How to Run Locally
1. Clone repo
2. Create MySQL database: `blood_donor_db`
3. Update `application.properties` with your credentials
4. Run: `mvn spring-boot:run`
5. Open: `http://localhost:8080/swagger-ui.html`
